Container

The .container class provides a responsive fixed width container.
The div with class "container" will have a fixed left and right margin and will not take the complete width of its parent or the viewport.

This is inside a Bootstrap container. Since we are using the class container, the div will not take complete width of it's parent.

Fluid Container

The .container-fluid class provides a full-width container which spans the entire width of the viewport.
The div with class “container-fluid” will take-up the complete width of the viewport and will expand or shrink when ever the viewport is resized.

This is inside a Bootstrap fluid container. Since we are using the class container-fluid, the below div will take-up complete width of the viewport and will expand or contract whenever the viewport is resized.
